# 🧠 OmniRAG (RAG ИИ-Агент)

**OmniRAG** — это автономная RAG-система, которая позволяет «общаться» с вашей персональной базой документов (PDF, DOCX, TXT, HTML, XML). Вы задаете текстовые вопросы, а локальная нейросеть сама находит нужную информацию в загруженных файлах, генерирует точный ответ и показывает ссылки на абзацы-источники.

### 🌟 Что реализовано:
- **Гибридный поиск**: Комбинация семантического (смыслового) и лексического (по ключевым словам) поиска гарантирует точное нахождение фактов.
- **Интерактивные цитаты**: Нейросеть прикрепляет кликабельные кнопки на источники ответа. При нажатии отображается оригинальный текст из документа, откуда был взят факт.
- **Полная приватность**: Работает на 100% локально через нейросеть Ollama без отправки ваших файлов в интернет.
- **Удобный веб-интерфейс**: Встроенный чат с боковой панелью управления для загрузки документов и настройки модели.
- **Умная обработка данных**: Специальный алгоритм чтения защищает длинные таблицы в документах от потери смысла.
- **Автоматическая адаптация оборудования**: Умеет работать как на обычных процессорах (CPU), так и на видеокартах (NVIDIA GPU).

## ⚡ Быстрый старт

Если вы скачали этот проект впервые, выполните эти 5 простых шагов, чтобы всё настроить:

### Шаг 1. Подготовка документов
Положите ваши PDF-статьи (например, исследования) в папку `data/` в корне проекта.

### Шаг 2. Запуск серверов
Откройте терминал в папке с проектом и запустите контейнеры. Выберите нужную команду в зависимости от вашего компьютера:

- **Если у вас НЕТ видеокарты NVIDIA (Ноутбук / CPU / Mac):**
  ```bash
  docker-compose up -d
  ```
- **Если у вас ЕСТЬ видеокарта (NVIDIA GPU на ПК/Сервере):**
  ```bash
  docker-compose -f docker-compose.yml -f docker-compose.gpu.yml up -d
  ```

### Шаг 3. Скачивание нейросети
Один раз скачайте языковую модель Qwen2.5 в контейнер:
```bash
docker exec ollama_db ollama pull qwen2.5:3b
```

### Шаг 4. Индексация базы знаний
Создайте умный индекс из файлов, которые вы положили в папку `data/`:
```bash
docker exec pharma_rag_ui python src/cli/ingest.py
```

### Шаг 5. Начало работы!
Откройте браузер и перейдите по адресу:
👉 **http://localhost:8501**

## Повседневные команды

**Обновление базы знаний (если добавили или удалили PDF):**
```bash
docker exec pharma_rag_ui python src/cli/ingest.py
```
*(После выполнения просто обновите страницу браузера - клавиша F5)*

**Остановка проекта:**
```bash
docker-compose down
```
